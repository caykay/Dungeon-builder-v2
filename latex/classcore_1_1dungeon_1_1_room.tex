\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classcore_1_1dungeon_1_1_room}{Room}} class. 

This class describes properties a room in a dungeon level. Each room has 4 corder/edges These edges are marked as Directions\+: north, south, east and west.
